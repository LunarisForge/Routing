`lunaris-forge/routing` directs incoming requests to the appropriate actions.

## Installation

To install the LunarisForge/Routing library, use Composer:

```sh
composer require lunaris-forge/routing
```

## Testing

Run the tests:

   ```sh
   vendor/bin/phpunit
   ```
